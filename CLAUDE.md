# CLAUDE.md

Guidance for Claude when working in this project (MRB Virtual Support).

## Front-end work

- **Always invoke the `frontend-design` skill before writing any front-end code.** Run it first, then write the code.
- Reference `brand_assets/` for the **logo, color palette, and brand guidelines** (`logo.png` and `brand.md`). Treat these as the source of truth for all visual decisions; do not invent colors, logos, or styling that conflict with them.
  - Primary `#123B6D` · Accent `#F4B24F` · Neutral background `#F8FAFC`.

## Design principles

- Keep designs **clean and modern**.
- **Never use Lorem Ipsum** or other placeholder filler text. Write realistic, contextually appropriate copy instead.
- **Never produce designs that look generic or AI-generated.** Avoid templated layouts, default component styling, and cookie-cutter patterns. Every design should feel intentional and distinctive, grounded in the brand.
